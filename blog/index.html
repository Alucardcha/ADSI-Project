¿Es HTML un lengüaje de programación?

HTML es un lenguaje de marcas de Hipertexto, así es el componente más básico de la Web. Define el significado y la estructura del contenido web, es decir, que se puede crear el esqueleto de una página web a través de etiquetas.

Ventajas y desventajas de Java
